# 🚀 GenAI Projects

This repository contains a collection of projects built while learning and experimenting with Generative AI concepts, including Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and AI-driven automation.

## 📌 Projects Included

### 1. AI-Powered HR Outreach Automation Agent

An automation system that generates and sends personalized cold emails to HR professionals using LLMs.

* Uses LangChain with Groq for email generation
* Parses structured outputs and stores them in JSON format
* Automates email delivery via a Python-based SendMail API
* Demonstrates prompt engineering, chaining, and real-world workflow automation

---

### 2. YouTube Chatbot using RAG

A chatbot that answers user queries based on YouTube video transcripts using a Retrieval-Augmented Generation approach.

* Extracts transcripts from YouTube videos
* Creates embeddings and stores them in a vector database
* Performs similarity search to retrieve relevant context
* Generates accurate answers using LLMs with augmented context

---

## 🛠️ Tech Stack

* Python
* LangChain
* Groq API
* Vector Databases (FAISS / Chroma)
* JSON
* YouTube Transcript API
